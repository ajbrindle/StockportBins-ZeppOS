This is a ZeppOS/Amazfit app to display the bin collection information for Stockport

An area id needs to be set via the app settings in the ZeppOS app. This can be found as follows:
1. Visit: stockport.gov.uk/find-your-collection-day
2. Enter post code, pick address and click through to the collection dates page.
3. Look at the URL in the browser address bar.
4. The area ID is the number located between the slashes after "show/".
   e.g. .../show/100011526336/39ROADNAME... has an area ID of 100011526336

The app will show the next bin collection date and which bins will be included. Swipe left to see the next collection date for any other bins.

The "Remind Me" button will set a watch reminder for 6:30pm the evening before the next collection.

This is an independently developed app and is not endorsed by Stockport MBC
